#### 软件架构
软件架构说明：服务端采用的是微服务架构，通过eureka进行注册管理，网关进行路由过滤。前端采用Vue框架，通过nginx代理转发。

###  实现功能
主要实现功能：门户网站，搜索课程，学习中心，用户中心，系统管理中心，教学管理中心。

项目需要安装的环境有：ElasticSearch、FastDFS(虚拟机)、nginx。
